# portboy

Easily call out to other languages from Elixir.

## Languages

See the READMEs in `./clients` for [examples](https://github.com/mvkvc/portboy/tree/main/clients).

- [X] [JS](https://www.npmjs.com/package/portboy)
- [ ] Python

## Installation

```elixir
def deps do
  [
    {:portboy, ">= 0.0.0"}
  ]
end
```
